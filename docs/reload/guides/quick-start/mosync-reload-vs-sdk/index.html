MoSync Reload Compared to the MoSync SDK

When you develop apps in HTML5/JavaScript, you have the choice of using MoSync Reload and/or the MoSync SDK.

MoSync Reload is targeted exclusively at HTML5/JS development, while the SDK is targeted at both C++ and HTML5/JS development. Reload programs runs inside the Reload Client app.

The MoSync SDK contains the Eclipse-based MoSync IDE, and produces native apps as its output.

A common workflow for developing apps in HTML5/JS is to develop and test the app in Reload, then use the SDK to build native packages for the different App Stores.
